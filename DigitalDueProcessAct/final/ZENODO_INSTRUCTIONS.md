How to publish a DOI for this packet using Zenodo

1. Create a GitHub release
- Tag a release (e.g., v1.0) and attach the `Minnesota_Packet_ZIP.zip`.

2. Link GitHub to Zenodo
- Create a free Zenodo account and connect your GitHub account (Settings → GitHub). Enable the repository for automatic archiving.

3. Create or trigger an archive
- When you create a GitHub release, Zenodo will archive it and mint a DOI. Alternatively, manually upload the ZIP to Zenodo and fill metadata (title, authors, abstract, keywords).

4. Choose license and DOI options
- Select CC BY 4.0 (we included `LICENSE` in the ZIP). Add citation information and contributors.

5. Publish and copy DOI
- After Zenodo publishes, copy the DOI and add it to `RELEASE_NOTES.md` and the cover sheet.

Notes:
- Zenodo provides a stable DOI and free hosting for research outputs. This makes the packet citable and preserves authorship provenance.
